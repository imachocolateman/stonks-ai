# stonks-ai

AI-driven stock market analysis and trading assistant.

## Overview

stonks-ai monitors and analyzes stocks to identify optimal buy/sell opportunities. Combines technical analysis, news sentiment, and predictive modeling to screen stocks and provide actionable insights.

## Features (Planned)

- **Multi-stock screening** - Track and analyze multiple tickers simultaneously
- **Technical analysis** - Chart patterns, indicators, and price action analysis
- **News sentiment analysis** - Real-time news impact assessment
- **Predictive modeling** - Historical data analysis for trend prediction
- **Moomoo API integration** - Direct connection to market data and trading platform

## Requirements

- Python 3.13+
- Moomoo API access

## Setup

1. Clone repository
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and configure API keys
4. Run application
